# richtext-editor

Halo 2.0 的默认富文本编辑器，基于 [Tiptap](https://github.com/ueberdosis/tiptap)。

## 开发环境

```bash
git clone git@github.com:halo-sigs/richtext-editor.git
```

```bash
pnpm install 

pnpm packages:build

pnpm example:dev
```
